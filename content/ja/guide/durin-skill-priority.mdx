<Section icon={Sparkles} title="形態変化：純粋か、闇か">

ドゥリンのゲームプレイは、元素スキルによって発動される**本質的な形態変化**のメカニズムを中心に展開されます。標準的なキャラクターとは異なり、彼のスキルには、次の30秒間の彼の状態を決定するフォローアップアクションがあります。この選択は、彼の元素爆発の挙動と、彼の固有天賦1で提供されるバフを完全に変更します。

<Steps>
  <Step num="1" title="最初のスキル発動">
    元素スキル**二元形態：収斂と分裂**を発動して、初期の範囲パイロダメージを与え、形態変化状態に入ります。
  </Step>
  <Step num="2" title="道を選択">
    **スキルボタンをもう一度押す**と**純粋の確定（白の形態）**に入り、**通常攻撃**を行うと**闇の否定（闇の形態）**に入ります。
  </Step>
  <Step num="3" title="エネルギー返還">
    いずれかの状態に入ると、ドゥリンは即座に固定エネルギーを回復します。この返還は6秒ごとに1回発生し、天賦レベルに応じてスケールします。
  </Step>
  <Step num="4" title="爆発を解き放つ">
    元素爆発は、選択した状態に基づいて**白き炎の龍**または**闇き腐敗の龍**の形態をとります。
  </Step>
</Steps>

  <Card title="白の形態（サポート）">

    - **最適なチーム**: 過負荷、燃焼、モノパイロ、風/岩サポート。

    - **最適なチーム**: 順方向蒸発、逆方向溶解、デュアルキャリー編成。
  </Card>
</CardGrid>

<Callout type="tip" title="形態の持続時間">
形態変化状態は30秒間持続し、爆発のクールタイムよりもかなり長いです。これにより、望む状態を失うことなく、ローテーションの早い段階で形態を切り替えることができます。
</Callout>

<Section icon={Swords} title="天賦のスケーリングとエネルギー管理">

**ドゥリンのスキル優先度**の重要な側面は、彼のスキルによって提供されるエネルギー返還です。天賦レベル10では、形態に入ると**固定エネルギー33**を回復します。彼の爆発コストは70なので、これにより実質的に必要なエネルギーは**37**に減少します。このメカニズムがあるため、高投資のプレイヤーは爆発とともにスキルもクラウンすることがよくあります。

| 天賦レベル | 返還エネルギー | 実質的な爆発コスト |
| :--- | :--- | :--- |
| レベル1 | 15.0 エネルギー | 55.0 エネルギー |
| レベル6 | 24.0 エネルギー | 46.0 エネルギー |
| レベル9 | 30.0 エネルギー | 40.0 エネルギー |
| **レベル10** | **33.0 エネルギー** | **37.0 エネルギー** |
| レベル13 (C5) | 42.0 エネルギー | 28.0 エネルギー |

ドゥリンの固有天賦4、**夜の如く形作られし混沌**は、彼の合計攻撃力に基づいて大幅なダメージブーストを提供します。攻撃力100ごとに、最初の10回の爆発ダメージが3%増加し、攻撃力2500で75%に上限があります。

| 攻撃力ステータス | 爆発ダメージ倍率 | 使用状況 |
| :--- | :--- | :--- |
| 1500 攻撃力 | +45% ダメージ | 低投資/サポート |
| 2000 攻撃力 | +60% ダメージ | 標準的なサブDPS |
| **2500 攻撃力** | **+75% ダメージ** | **最適なスケーリング上限** |

<Callout type="success" title="スケーリングの洞察">
ドゥリンは攻撃力2500の上限まで攻撃力に大きく依存するため、しきい値を下回っており、ベネットや旧貴族のしつけのような外部からの攻撃力バフがない場合、**攻撃力%杯**はパイロダメージ杯と競合することがよくあります。
</Callout>

</Section>
