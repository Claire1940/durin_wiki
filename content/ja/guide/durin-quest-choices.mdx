|                 | 船の方へ向かう（隠し） | グッド | 風の道の近くにある「ドゥリン号」と2回インタラクトします。この隠し選択肢は放浪者とミニドゥリンのテイワットへの旅の記憶を呼び起こし、別のグッドエンディングと任務の進行に繋がります。 |

<Section icon={Swords} title="第5部：運命の解法 - 最終的な決断">
第5部「運命の解法」は、ドゥリン伝説任務のフィナーレを飾り、最も重要な選択を提示します。ここでは、あなた、アルベド、放浪者、そしてドゥリンが絵本の中に入り、ドゥリンの核心的な問題に取り組みます。

最初の決断は、邪龍の心臓に関するものです：

| :-------------- | :----------------------------------- | :----------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **グッドエンディング** | 邪龍の心臓と融合する | グッド | アルベドが邪龍の心臓を使ってドゥリンの擬似的な体を作るポジティブな結果に繋がります。これは魔神任務の出来事を反映しており、ドゥリンの新しい始まりへの重要なステップです。 |
|                 | 小さな龍と対話する（隠し） | グッド | 心臓について決めるシーンで、絵本の左上隅にいるミニドゥリンを見つけてインタラクトします。この隠しアクションは自動的にグッドエンディングをトリガーし、ドゥリン自身の意志を示します。 |
| **バッドエンディング** | 邪龍の心臓を破壊する | バッド | 心臓を破壊することを選択すると、アルベドはミニドゥリンの擬似的な体を作ることができず、任務はネガティブな解決を迎えます。この選択はドゥリンが肉体を得ることを妨げます。 |

邪龍の心臓に関する決断の後、グッドルートを選んだ場合は、暗闇を食い止めることに関連する別の選択肢に直面します。

| 選択のカテゴリー | 具体的な選択肢 | 結末のタイプ | 説明 |

| **バッドエンディング** | 屈する / 時間切れ | バッド | 選択には59秒の制限時間があります。「暗闇を食い止める」を選択できなかったり時間切れになったりすると、ドゥリンは暗闇に飲み込まれ、悲劇的な結末を迎えます。 |
|                 | シムランカの城に戻る | バッド | 第4部のバッドエンディングと同様、ドゥリンがシムランカに退却し、彼の物語が消えていく結果になります。これは運命に立ち向かうのではなく、回避することを表しています。 |
