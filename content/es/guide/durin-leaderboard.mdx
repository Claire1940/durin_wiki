### Rotaciones de Combate Principales de Durin

| Forma       | Secuencia de Activación          | Frecuencia de Explosión | Capacidad de Reacción         |
| :--------- | :--------------------------- | :-------------- | :-------------------------- |
| **Negación** | Habilidad → Ataque Normal → Explosión | Cada 1.25s     | Cada dos golpes (Vaporizar/Derretir) |
| **Pureza** | Habilidad → Habilidad → Explosión        | Cada 1s        | Cada dos golpes (Pyro en área)  |

<Steps>
  <Step num="1" title="Iniciar Forma de Negación">
    Para entrar en la forma de Negación, usa la Habilidad Elemental de Durin, seguida de un Ataque Normal, y luego activa su Habilidad Definitiva. Esta forma es excelente para una aplicación Pyro consistente fuera del campo y para activar reacciones.
  </Step>
  <Step num="2" title="Optimizar Forma de Pureza">
    Para la forma de Pureza, activa la Habilidad Elemental de Durin dos veces, luego usa su Habilidad Definitiva. Esta forma ofrece una tasa de golpe de Explosión más rápida y una aplicación Pyro en área más amplia, ideal para ciertas composiciones de equipo.
  </Step>
  <Step num="3" title="Maximizar Ventanas de Reacción">
    Programa las aplicaciones elementales de tus otros personajes para que coincidan con los golpes de la Explosión de Durin, especialmente para las reacciones de Vaporizar y Derretir, para maximizar la producción de daño.
  </Step>
</Steps>
